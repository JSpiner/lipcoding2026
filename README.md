# 말그림 (MalGrim) · SpeakDraw

말그림은 음성으로 다이어그램을 만들고 수정하는 웹 앱입니다.  
회의나 설계 논의 중에 키보드/마우스 조작 없이, 말로 설명한 내용을 실시간 다이어그램으로 시각화하는 것을 목표로 합니다.

## 프로젝트 한 줄 소개

- 음성 명령을 해석해 다이어그램 IR(JSON)을 편집하고, Mermaid로 즉시 렌더링하는 생산성 도구
- 타깃 사용자: 개발자/아키텍트, PM/기획자
- 핵심 가치: "그리기 작업"을 "말하기"로 전환해 문서화 속도와 논의 효율 개선

## MVP 범위

- 음성으로 플로우차트 생성
- 음성으로 노드/연결/라벨 수정
- 플로우차트/시퀀스 다이어그램 지원
- 다이어그램 타입 전환(플로우차트 ↔ 시퀀스)
- Mermaid 텍스트/이미지 내보내기
- 텍스트 입력 폴백(마이크 불가 환경 대비)

## 해커톤 맥락

이 프로젝트는 "천하제일 입코딩 대회 2026" 출품을 전제로 기획되었습니다.

- 도전 과제: 개인 생산성 향상 앱
- 제한 시간: 4시간(입코딩)
- 기술 형태: 웹 앱
- 배포: Azure 클라우드

## 대회 주요 규칙 요약

아래는 ../lipcoding-competition-2026 문서 기준의 핵심 규칙 요약입니다.

1. 시작 전 60분(사전 준비 시간)에는 키보드/마우스 사용이 허용됩니다.
2. 입코딩 본 시간에는 VS Code + GitHub Copilot(+ Copilot CLI) 중심으로 진행하며, 다른 개발 도구 사용은 제한됩니다.
3. 본 시간에는 키보드/마우스(트랙패드) 직접 코딩이 금지됩니다.
4. 예외적으로 음성 입력 트리거, 제출 처리, 장애 대응 시 운영진 승인 하에 입력장치 사용이 허용됩니다.
5. 대회 종료 30분 전부터는 키보드/마우스 사용이 다시 허용됩니다.
6. 종료 시점까지 앱을 제출하지 못하면 자동 탈락입니다.
7. 필수 구현/운영 요건: 웹 앱, Copilot SDK 사용, Azure 배포.
8. 규칙 위반 시 옐로우/레드/강퇴 페널티가 적용되며, 누적 경고 시 실격될 수 있습니다.
